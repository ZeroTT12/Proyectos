🏠 Housing Price Prediction – Colorado (2026)

This project analyzes housing prices in the Colorado real estate market using machine learning models. The objective is to identify key price determinants and evaluate predictive performance using both structured data and listing descriptions.

The dataset includes variables such as square footage, bedrooms, bathrooms, and garage capacity. In addition, NLP techniques were applied to extract keywords from property descriptions (e.g., luxury, modern, mountain view), capturing qualitative features.

Two models were implemented:

Random Forest
XGBoost

Results show strong predictive performance, with XGBoost slightly outperforming Random Forest (R² ≈ 0.58). The most important variables are:

Property size (sqft)
Number of bathrooms
Number of bedrooms
Text-based indicators (views, amenities)

Overall, the findings highlight the importance of combining structured data with textual information to improve housing price predictions.
