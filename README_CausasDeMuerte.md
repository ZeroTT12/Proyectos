Infant Mortality Dashboard – Americas
Overview

This project analyzes infant mortality rates and causes of death in countries of the Americas using data from the World Health Organization (WHO). The dashboard allows users to explore mortality indicators, age distribution of deaths, and the main causes of infant mortality.

Data Source

World Health Organization (WHO) – Global Health Observatory

Mortality data per 1,000 live births

Countries from the Americas region

Tools Used

Excel – Initial data review

Power Query – Data cleaning and transformation

Power BI – Data modeling and visualization

Dashboard Features

Mortality indicator (deaths per 1,000 children)

Distribution of deaths by age group

Ranking of main causes of death

Interactive filters by country and year

Purpose

To provide a visual and interactive analysis tool to better understand patterns and causes of infant mortality in the Americas.
